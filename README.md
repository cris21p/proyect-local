# Proyecto Local

Este proyecto es una aplicación web sencilla.

## Estructura del proyecto
- `index.html`: Página principal.
- `styles.css`: Estilos de la aplicación.
- `font-family.css`: Fuentes personalizadas.
- `scrip.js`: Lógica y funcionalidades en JavaScript.
- `imagenes/`: Carpeta de imágenes.

## Instalación
No requiere instalación. Solo abre el archivo `index.html` en tu navegador.

## Uso
1. Clona o descarga el repositorio.
2. Abre `index.html` en tu navegador.

## Tecnologías utilizadas
- HTML
- CSS
- JavaScript
